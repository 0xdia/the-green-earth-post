Technical Architecture: Powering The Green Earth Post

June 01, 2025 posted by The Green Earth Team

Eco-Friendly Cloud Infrastructure

At The Green Earth Post, we've built a sustainable technical architecture that minimizes our carbon footprint while maximizing performance. Our infrastructure leverages Terraform-managed AWS resources combined with manually configured services for optimal efficiency.

Architecture Overview

Our solution uses a multi-tier architecture with clear separation of concerns:

Key Components

Terraform-Managed Infrastructure:
• VPC with public and private subnets
• EC2 instance running our API server
• Security groups controlling network access
• IAM roles with least-privilege permissions
• S3 bucket for static asset storage

Manually Configured Services:
• RDS MySQL database for content storage
• Additional S3 bucket for user uploads
• CloudFront CDN for global content delivery

Sustainability Features

• Auto-scaling groups to minimize idle resources
• EC2 spot instances for reduced energy consumption
• S3 Intelligent Tiering for storage optimization
• Database read replicas to distribute load efficiently
• CloudFront edge caching to reduce data transfer
